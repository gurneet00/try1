Mobile System Monitor - Installation Instructions

Note: The direct APK download is currently unavailable. Please follow these instructions to install the mobile monitoring app on your Android device.

Option 1: Build from Source

To build the app from source, you'll need:

  • Android Studio
  • Basic knowledge of Android development
  1. Clone or download the project files

    The mobile app source code is available in the mobile_apk_project directory.

  2. Open the project in Android Studio

    Start Android Studio and select "Open an existing Android Studio project". Navigate to the mobile_apk_project directory and select it.

  3. Build the APK

    Select Build > Build Bundle(s) / APK(s) > Build APK(s) from the menu.

  4. Install on your device

    Transfer the generated APK to your Android device and install it.

Option 2: Use a Third-Party Monitoring App

You can use any of these third-party system monitoring apps:

  • Stark Fitness - System Monitor
  • System Monitor
  • System Monitor - Battery, CPU, Device Info

These apps provide similar functionality to our custom app.

Configuring the App

Once you have installed a monitoring app, configure it with:

  • Server URL: http://YOUR_SERVER_IP:5000
  • Authentication Token: default_token_change_me (or your custom token)

Need Help?

If you're experiencing issues with the installation, please contact our support team for assistance.
